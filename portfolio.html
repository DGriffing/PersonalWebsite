Clean the Realm | March 2015

What is it: An application prototype to help roommates keep their living space clean and manage household chores using gamification techniques.

Why I'm proud of it: It was a quarter long group project and our team walked away with some awesome deliverables and learned a great deal about design thinking and working as a group for an extended period. At the time, I only knew Java, and was thrilled to be able to create so much with only that (hence prototype).

Skills Learned: Research Interviewing | Prototyping | User Testing | Creating Design Specs & Process Books

Dawg Coffee | September 2015

What is it: Dawg-Coffee is a semantically rich website for a coffee venue near the University of Washington (Go Huskies!).

Why I'm proud of it: This was my first time creating a website, and I succeeded in creating an aesthetically pleasing Website, learning to link pages efficiently, and overcoming the struggles of svg attribute editing.

Skills Learned: HTML | CSS | Mobile First | Bootstrap

Police Shooting Map | October 2015

What is it: A dynamic map showing instances of police shootings (from deadspin database). Colors indicate whether victims were armed or unarmed, and users can filter instances by victim age.

Why I'm proud of it: This was the first API I had to navigate with limited assistance, and I did so successfully. Also proud of my algorithm for calculating incident outcomes (hit, killed, unknown) which prompted a few classmates to say "damn, that's cool."

Skills Learned: Leaflet | JavaScript | jQuery

Magic the Gathering: Card Review | October 2015

What is it: A website to review your favorite Magic The Gathering cards (just kidding, it only lets you review one... for now). It interacts with its Parse database, to give a list of previous reviews, which users may then interact with, voting up or down for usefullness, or deleting.

Why I'm proud of it: Understanding some of the complexities of working with an online database and its API was very challenging for a first time user, but I was determined to make it work, and I did! In addition, I was able to understand what was going on enough so that I could help out fellow classmates, which is one of the most rewarding experiences for me.

Skills Learned: Parse | Database Management | Code Injection Protection
